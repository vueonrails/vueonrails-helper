# Vue on Rails helper

A javascript helper for [Vue on Rails](http://vueonrails.com)

It enables Specific-page Vue, passing data into Vue component and other utilities for your modern Vue on Rails project.

## Usage

```javascript
require('vueonrails')
```

```javascript
import {} from 'vueonrails'
```
